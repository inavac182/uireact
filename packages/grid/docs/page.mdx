import { UiText } from '@uireact/text';

import { UiGrid, UiGridItem } from '@uireact/grid';
import packageJson from './package-metadata.json';
import { Metadata } from '@uireact/docs-tools';

# UiGrid & UiGridItem

<Metadata packageName='grid' packageJson={packageJson} description='Structure and organize the UI using a grid column' includeInformation />

## Usage

### 1. Using just UiGrid

```jsx live scope={{UiGrid, UiCard, UiText}}
  <UiGrid cols={3} rows={2} colsGap="four" justifyItems="center">
    <UiCard fullWidth fullHeight>
      <UiText>Item 1!</UiText>
    </UiCard>
    <UiCard fullWidth fullHeight>
      <UiText>Item 2!</UiText>
    </UiCard>
  </UiGrid>
```

### 2. Using UiGrid with UiGridItem

```jsx live scope={{UiGrid, UiGridItem, UiCard, UiText, UiViewport}}
  <UiGrid cols={3} rows={2} colsGap="four">
    <UiViewport criteria="l|xl">
      <UiGridItem cols={2}>
        <UiCard>
          <UiText>Item 1!</UiText>
        </UiCard>
      </UiGridItem>
    </UiViewport>
    <UiGridItem placeSelf="stretch" rows={2}>
      <UiCard>
        <UiText>Item 2!</UiText>
        <UiText>Yeah, 2 rows is nice!</UiText>
      </UiCard>
    </UiGridItem>
  </UiGrid>
```

In this case as we are using the UigridItem we are able to manipulate each grid item on its own.

Also notice that in Item 1 we are making use of the [UiViewport](./packages-foundation-docs-viewport) to render/remove it depending on the breakpoint

### 3. Using GridBreakpointsDistribution prop

This prop give you the ability to set a number of cols/rows depending on the breakpoint.

```jsx live scope={{UiGrid, UiCard, UiText}}
  <UiGrid cols={{ small: 1, medium: 2, large: 3, xlarge: 3 }} rows={2} colsGap={"five"} rowsGap={"six"}>
    <UiCard>
      <UiText>Item 1!</UiText>
    </UiCard>
    <UiCard>
      <UiText>Item 2!</UiText>
    </UiCard>
    <UiCard>
      <UiText>Item 3!</UiText>
    </UiCard>
  </UiGrid>
```

### 2. Using custom UiGridItem col span

```jsx live scope={{UiGrid, UiCard, UiText, UiGridItem}}
  <UiGrid cols={4} colsGap="three">
    <UiGridItem cols={2}>
      <UiCard>
        <UiText>Item 1!</UiText>
      </UiCard>
    </UiGridItem>
    <UiGridItem placeSelf="stretch" cols={2} startingCol={3} startingRow={2}>
      <UiCard>
        <UiText>Item starting on the 3rd col and 2nd row</UiText>
      </UiCard>
    </UiGridItem>
  </UiGrid>
```
