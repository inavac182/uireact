# UiProgressIndicator

<Metadata packageName='progress-indicator' packageJson={packageJson} description='Used to represent the progress in a multi step process' includeInformation />

## Usage

```jsx live scope={{UiProgressIndicator, UiProgressIndicatorItem, UiText}}
<>
  <UiProgressIndicator>
    <UiProgressIndicatorItem>
      <UiText>Step 1</UiText>
    </UiProgressIndicatorItem>
    <UiProgressIndicatorItem>
      <UiText>Step 2</UiText>
    </UiProgressIndicatorItem>
    <UiProgressIndicatorItem>
      <UiText>Step 3</UiText>
    </UiProgressIndicatorItem>
  </UiProgressIndicator>
</>
```
