# UiCard

<Metadata packageName='card' packageJson={packageJson} />

> Component to render content, it has built in styling and props to make it flexible. It also extends motion.div so you can pass through
any prop for motion animations.

## Installation

```js
  npm i @uireact/foundation @uireact/card
```

## Usage

### UiCard with built-in animation

In this example we are using a built-in animation passing the prop `animation`. There are several built-in options to choose from, you can visit the [built-in animations doc](/docs/built-if-animation) to learn more

```jsx live scope={{UiCard, UiHeading, UiText}}
  <UiCard animation='fadein'>
    <UiHeading>ðŸ¥µ Card header</UiHeading>
    <UiText>This is some content</UiText>
  </UiCard>
```

### UiCard with click handler and motion animation

In this example we are passing the prop `whileHover` and `whileTap` to perform a scaling animation when any of these events happens.

You can learn more about it in the [framer motion docs](https://www.framer.com/motion/component/)

```jsx live scope={{UiCard, UiHeading}}
  <UiCard motion={{whileHover:{ scale: 1.2 }, whileTap:{ scale: 0.8 }}} clickHandler={(identifier) => console.log(`Card ${identifier} clicked`)} identifier="My-ID">
    <UiHeading>Click me!</UiHeading>
  </UiCard>
```

### UiCard with only content

```jsx live scope={{UiCard, UiHeading, UiText}}
  <UiCard>
    <UiHeading>Card header</UiHeading>
    <UiText>This is the content</UiText>
    <UiText size="xsmall">This is the footer</UiText>
  </UiCard>
```

### UiCard as a link

```jsx live scope={{UiCard, UiText}}
  <UiCard link="https://www.uireact.io" bordered>
    <UiText>Open UiReact docs</UiText>
  </UiCard>
```

### UiCard with image content

```jsx live scope={{UiCard, UiHeading, UiText, UiSpacing}}
  <UiCard padding={null}>
    <img
      src="https://upload.wikimedia.org/wikipedia/commons/e/e1/Colima_360.jpg"
      style={{ width: '100%', borderRadius: '10px 10px 0px 0px' }}
    />

      <UiHeading>Card header</UiHeading>
      <UiText>This is the content</UiText>
```

### UiCard outlined

```jsx live scope={{UiCard, UiHeading, UiText, UiSpacing}}
  <UiSpacing margin={{ block: 'three' }}>
    <UiCard weight="10" category="positive" styling="outlined">
      <h3>Card header</h3>
      <p>This is the content</p>
      <UiText size="small">This is the footer, using UiText styling is different.</UiText>
    </UiCard>
  </UiSpacing>
```

The outlined card will give some basic coloration to texts, although if you use any text component
the styling will be overwritten and has to be manually set up on each text component.
