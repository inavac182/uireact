# UiIcon

<Metadata packageName='icons' packageJson={packageJson} description='Component used to render icons in a web application' includeInformation />

## UiIcon

```jsx live scope={{UiIcon}}
  <UiIcon icon="Search" size="xlarge" category="positive" />
```

## UiIcon animated with motion props

This prop will animate the icon container it won't animate the actual SVG paths, for that we are working on a new package of animated icons 
as that requires some rework on the icons itself.

```jsx live scope={{UiIcon,UiReactViewShaking}}
  <UiIcon icon="BellRing" size="xlarge" category="positive" motion={UiReactViewShaking} />
```

You can see the docs for [@uireact/framer-animations](./framer-animations) package to see the full list of available animations.

## UiIcon with weight

Default is weight 100

```jsx live scope={{UiIcon}}
<>
  <UiIcon icon="Search" size="xlarge" category="tertiary" weight="10" />
  <UiIcon icon="Search" size="xlarge" category="tertiary" weight="50" />
  <UiIcon icon="Search" size="xlarge" category="tertiary" />
  <UiIcon icon="Search" size="xlarge" category="tertiary" weight="150" />
  <UiIcon icon="Search" size="xlarge" category="tertiary" weight="200" />
</>
```

## UiIcon with inverse coloration

```jsx live scope={{UiIcon}}
  <UiIcon icon="Search" size="xlarge" category="tertiary" inverseColoration />
```

## UiIcon with default coloration

```jsx live scope={{UiIcon}}
<>
  <UiIcon icon="Search" size="xlarge" category="tertiary" coloration="dark" /> Always render DARK coloration <br />
  <UiIcon icon="Search" size="xlarge" category="tertiary" coloration="light" /> Always render LIGHT coloration
</>
```
